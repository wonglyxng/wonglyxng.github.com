关于独立游戏的一点思考

Posted on 2022-03-17

游戏与即时通讯互通设定

聊天功能

游戏内可以通过即时通信进行交流，就算退出游戏，游戏内的好友或有你的通讯的方式的角色也可以给你发送信息

  • 邮箱
  • qq
  • 微信
  • 其他

订阅功能

游戏内可以对一些重大事件或公告进行订阅，游戏会通过玩家预留通讯方式进行推送

  • 邮箱
  • qq
  • 微信
  • 其他

游戏世界进化设定

游戏内NPC都有自己的人生轨迹，有自己的命运，进行不同的生活，玩家与NPC共同推动游戏世界的进化
NPC和玩家具有相同的世界背景，NPC可以给玩家发放任务，玩家也可以给NPC发放任务，NPC也可以升级成长
游戏世界初始诞生NPC，最初NPC进行繁衍，推动世界进化，这部分为和平演化阶段，然后玩家降临，玩家与NPC竞争世界运气，决定世界进化方向，这部分为竞争演化阶段

婚姻系统

玩家与玩家之间可以结婚，玩家与NPC之间可以结婚，NPC与NPC之间也可以结婚，结婚条件：好感度，亲密值

礼物系统

玩家与玩家之间可以互送礼物，玩家与NPC之间也可以互送礼物，NPC与NPC之间也可以互送礼物

交易系统

玩家与玩家之间可以交易，玩家与NPC之间也可以交易，NPC与NPC之间也可以交易

宠物系统

所有诞生灵智的生物都可以与之签订契约，然后成为你的宠物

装备系统

装备可以自己打造，使用不同材料可以打造不同品质装备，世界会诞生各种奇珍异宝，根据世界进化的不同会诞生不同品质材料

功法系统

功法由NPC和玩家创造，最初诞生的NPC会创建基本功法，玩家可以对功法进行改进，也可以创造功法，玩家创造的功法得到世界认可后将可以存档供其他人学习，可以设置学习条件

境界系统

炼器系统

修炼系统

师徒系统

好友系统

家族系统

帮派系统

玩家和NPC都可以创建

宗门系统

玩家和NPC都可以创建，条件：可以创造功法，玩家和NPC都可以加入，然后学习功法，宗主创造的功法可以设定是否只能本宗门成员学习

三界系统

人间， 地府， 仙界

任务系统

奇遇，冒险，副本，活动，竞技

种植，采矿，寻宝，钓鱼，炼金

ubuntu20.4编译opencv-c++

Posted on 2021-12-22 Edited on 2021-12-23

1.创建opencv_build目录
$ cd ~/work/cppspace
$ mkdir opencv_build && cd opencv_build

2.下载opencv和opencv_contrib源码
$ git clone https://github.com/opencv/opencv.git
$ git clone https://github.com/opencv/opencv_contrib.git

3.创建build目录
$ cd opencv
$ mkdir -p build && cd build

4.构建opencv
$ cmake -DCMAKE_BUILD_TYPE=RELEASE -DCMAKE_INSTALL_PREFIX=/usr/local -DINSTALL_PYTHON_EXAMPLES=ON -DBUILD_EXAMPLES=ON -DOPENCV_GENERATE_PKGCONFIG=ON -DOPENCV_EXTRA_MODULES_PATH=~/work/cppspace/opencv_build/opencv_contrib/modules -DINSTALL_C_EXAMPLES=ON ..

5.查看cpu核数
$ nproc
8

6.开始编译
$ make -j8

7.安装opencv
$ sudo make install

8.验证安装
c++:
$ pkg-config –modversion opencv4
4.5.5

python:
$ python3 -c “import cv2; print(cv2.version)”
4.3.0

关于神经元结构的一点思考

Posted on 2021-11-27

1.结构的多样性

目前神经元大多数结构都是单一的，那么如果各个神经元具有多种形式是否可行，不同形式能够体现更复杂的多样性

2.哺乳动物神经元类型

1.单极神经元

2.双极神经元

3.假单极神经元

4.多极神经元

形态学相似的神经元倾向于集中在神经系统的某一特有区域，并具有相似的功能，但神经系统各区域的神经元形状却千差万别。
