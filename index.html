ubuntu20.4编译opencv-c++

Posted on 2021-12-22

1.创建opencv_build目录
$ cd ~/work/cppspace
$ mkdir opencv_build && cd opencv_build

2.下载opencv和opencv_contrib源码
$ git clone https://github.com/opencv/opencv.git
$ git clone https://github.com/opencv/opencv_contrib.git

3.创建build目录
$ cd opencv
$ mkdir -p build && cd build

4.构建opencv
$ cmake -DCMAKE_BUILD_TYPE=RELEASE -DCMAKE_INSTALL_PREFIX=/usr/local -DINSTALL_PYTHON_EXAMPLES=ON -DBUILD_EXAMPLES=ON -DOPENCV_GENERATE_PKGCONFIG=ON -DOPENCV_EXTRA_MODULES_PATH=~/work/cppspace/opencv_build/opencv_contrib/modules -DINSTALL_C_EXAMPLES=ON ..

5.查看cpu核数
$ nproc
8

6.开始编译
$ make -j8

7.安装opencv
$ sudo make install

8.验证安装
c++:
$ pkg-config –modversion opencv4

python:
$ python3 -c “import cv2; print(cv2.version)”

1.单极神经元

2.双极神经元

3.假单极神经元
